イベント中止と保険

長い時間を掛けて取り組み、万全の体制で当日を迎えても、イベントを中止せざるをえない場面というのは発生し得る。

しかし、多くの場合イベントの準備は「当たり前に開催されるもの」という前提で進めてしまい、中止を想定してない場合が多い。

イベントを開催する場合は、必ず「中止する場合はどうなるか」を考えておく必要がある。

# 中止の要因

最も多い要因は台風だ。

交通も麻痺し、会場にたどり着けなかったり、危険を伴う場合、無理に開催すると事故などにつながる可能性もある。

他にも、コロナの流行による緊急事態宣言で、外出の自粛などを要請された場合、開催自体が無理になる。

天気が良かろうと、規模が小さかろうと、中止をしなくてはならない事態は、どんなイベントにもありえるのだ。

# 中止ポリシー

中止をする場合、「どう中止するのか」はあらかじめ考えておく必要がある。

## 中止の決断と告知

天候に起因する場合、例えば朝の 10:00 に開場だが、前日夜の時点で台風が来ると予報されている場合を考えよう。

もし前日中に何も通知が無ければ、 10:00 に間に合うように開場に向かうべきなのかどうなのか、参加者は自己判断を迫られることになる。

特に遠方からの参加者は、前日入りして一泊してから参加などもある。

イベントが開催されるなら行きたいが、中止するのであれば、諸々をキャンセルしたい。

こうしたことを考えると、中止するのであればその決断は早いほうが良く、それ以上に「いつまでに中止の判断がされるのか」が事前にわかっている方が参加者が動きやすくなるのだ。

例えば、

「天候などの要因で開催が難しそうだと判断された場合は、前日夜 21:00 までに判断して通知する。

ただし、その時点で開催判断でも、当日の状況をみて中止に覆る可能性はある。その場合は、朝 8:00 までに判断し通知する。」

というポリシーがあるだけでも、参加者はかなり動きやすくなるのだ。

主催としてはギリギリまで判断を延期したい気持ちはわかるが、ダラダラ引き伸ばしても、スタッフや登壇者含めどう動いて良いのかわからず

結果として全員開場まで来たのに結局中止、や、開催にしたのに誰も開場に来ないといった、最悪の状態になりえる。

きちんとポリシーを決め、少しでも「難しいかも」と思ったら、すっぱり中止にするのも主催の重要な責任だ。

## 中止の告知方法

基本的には SNS などでの告知になるだろうが、緊急性の高いものは直接参加者に届く方法で通知したい。

そして、参加者もなるべく新鮮な情報を取得したいが、どのチャネルで連絡が来るのかわからない、といった状況では取りこぼしもあり得る。

開催する場合は、「中止の場合にはこの時間までに方法で連絡をする」というチャネルの明示もおこなっておきたい。

## スポンサーへの説明

スポンサーを集め、資金を受け取っているような場合、イベントを中止したらどうするのか、といったことは事前に決めておかないとトラブルになる。必要によってはスポンサー元に実際に出向いて説明そしてお詫びする必要があるだろう。

基本的に、イベントは開催までに資金をほとんど使っているだろう。つまり、中止になったからといってスポンサーに対して返金できることは少ない。

したがって、スポンサーと契約する際に、必ず「イベントは中止になり得るが、その場合も返金はできない、もしくはスポンサー金額80%返金」といったことを明示した「イベント中止ポリシー」を制定して、了承の上資金の提供を受ける必要がある。

# 興行中止保険

イベントの中止によって、経済的な損失が想定されるような構成にしている場合は、保険に入ることを推奨する。

興行中止保険は、その名の通りイベントの主催が入り、中止になった場合に保険金が降りることで損失を補填する商品だ。

[【興行中止保険】イベントの開催に関連して発生した事故による損害を補償する保険はありますか？ | よくあるご質問（FAQ） | 東京海上日動火災保険](https://faq.tokiomarine-nichido.co.jp/faq/show/2158?category_id=332&site_domain=default)

台風などはカバーされる一方で、コロナなどの感染症は対象外になっていたり、その上でそこそこ保険料も高いので加入するかどうかは検討の余地があるだろう。

興行中止保険でイベントの損失に対して保証される金額は、保険業界内では 80〜90 %となっている。損失全額は保証されない。例えば、損害対象金額 1,800 万で損害カバー率 80% の場合は 1,440 万が保険が降りて損失(支出)に対して補填することができる。

また、保険対象期間が存在する。加入(締結)後からイベント開催まで保険対象とするのが一般的である。カンファレンスの場合、チケット販売をカンファレンス開催までギリギリまで販売しているケースがある。保険保証対象期間のことを考えると、3 週間〜 1 ヶ月前で行うのが妥当であるが、締結後に支出があった場合は差分支出は保険対象にならないので、保険契約する際はそれを考慮する必要がある。

他の保険として、会場内での事故や備品等の物損の保険となる「施設賠償責任保険」はそこまで保険料も高くなく加入をおすすめできる。

## Vue Fes Japan 2019 の事例

実際に興行中止保険が使われた事例として、 Vue Fes Japan 2019 (以降 Vue Fes 略) がある。

このイベントは、スタッフ、スポンサーも合わせて 1,400 名以上の来場を見込む大型カンファレンスだ。

海外スピーカーの渡航・宿泊費をはじめ、ランチやアフターパーティー、印刷、同時通訳、各種運営など、多額の費用が発生していた。

ところが、開催日の数日前に、史上最強級とされる台風の接近が報道され、 JR の計画運休が発表されるなど、交通の麻痺などが予想された。

しかし、 Vue Fes は、東京海上日動の興行中止保険に入っていたため、最終的には費用の一部が保証された。全額ではなく、また中止に当たって追加で発生した費用(ごみ処理費、郵送費など)もあるため、赤字自体は発生した形だ。

その費用で、参加者のチケット代の全額返金対応を行い、スポンサーについては協賛金の全額返還はできず、一部は負担してもらう形になった。

## 保険適用判断のタイミング

Vue Fes でのケースで難しかったのは、保険適用の判断タイミングだ。

このときの保険適用の条件は、以下のいずれかが満たされた場合に、一部を補填するものだった。

- 当日、開催地より半径 100 km 以内に開催に影響を与える自然現象(今回の場合は台風)がある
- または開催地への交通機関が運休などで機能しなくなった場合

そして、この条件が確認されるのが「開催当日」なので、開催当日にならないと保険が降りるかはわからない。つまり、前もって中止の判断をしても、当日条件が変わった場合保険が降りず、全ての費用を Vue Fes が負担することになってしまう。

そこで当初 Vue Fes は、保険適用の判断を待ち、当日の朝に開催可否を告知する方針を取っていた。

しかし、最終的には「金銭リスク」よりも「参加者の安全」を優先する判断をし、前日の次点で中止を発表。

保険の規約上、「代替の開催」がある場合は対象外になるため、コンテンツを使い回す別イベントなどは開催できない。したがって、完全に中止のまま 2019 年を終えることとなった。

## 保険だけに頼らない中止計画

保険の商品も当時より増えたが、どの商品でも基本的に「全額保証」を得るのは難しい。つまり、中止をせざるを得ない場合は、一定の損失が発生することは、開催するうえで織り込んでおく必要がある。

特に、プール金などのない運営母体が、予算の大きいイベントを行い、中止になって保険も降りないとなると、そうとうの負債が発生してしまうことになる。

その場合は、チケットのスポンサーを取るにあたり、事前に「中止時の対応」について明示しておき、返金が無いことなどを明示しておくといった方法もある。

中止時に発生するリスクをあらかじめコントロールしておくことは、不足の自体が発生した際に、迅速に中止の決断をするといった行動につながる。

参加者にとって、中止はチケット代だけでなく、交通費・宿泊費などにも影響するが、もし早めに中止がわかれば、キャンセル料が 100% から 50% に減らせるといった影響もある。

また、参加者側も「旅行キャンセル保険」や「カード付帯保険」などで、中止時の出費を補填する自衛も可能な場合がある。来場者に対する返金を行わないチケットを販売する際は、そうした手段を案内するという手もあるだろう。

## 手に余る規模にしない

参加者の見込めるテーマでカンファレンスを企画すれば、ある程度のスポンサーを集めることは出来てしまう。

そこで集めた資金を用いて、大きめの会場を借り、海外からエンジニアを招いて渡航費を出し、同時通訳を付け、豪華なケータリングを取り、ロゴ入りノベルティを作り、ドメインを取ってサイトを公開し、強気のチケットで多くの参加者を集めることは、不可能な話ではない。

しかし、全く初めての状態で、いきなりそうした大規模なカンファレンスを開催しようとすると、まさしく「不足の事態」が発生した際に、それらをカバーし切る難易度も変わって来る。

当日、台風や大雪や疫病やテロなど、不足の事態で開催中止になったが、保険もなく、スポンサーやチケットも全額返金する必要が出れば、大きな赤字を抱えることになる。法人がなければ、それは主催個人が多額の負債を抱えることになるのだ。

法人でイベントを定期的に開催するのであれば、チケットやスポンサーで少し余裕を持って集め、開催後の余りをプールしておくこともできる。それは、次の開催の元でや、何かあったときの補填に回すこともできる。

資金以外にも、スタッフの練度や、コミュニティの成熟が伴わない状態で、いきなり大きなカンファレンスをやるのは難しい。最初はあまり大きくしすぎないところから始め、組織自体の練度を上げながら、徐々に規模や予算を増やしていくのが、望ましい姿だろう。
