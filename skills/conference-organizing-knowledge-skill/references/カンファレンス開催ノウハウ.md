# 開催ノウハウの継承

コロナ禍により、それまで盛り上がっていたイベントが軒並み開催を止めることになりました。

そこから、また新しくイベントが開催され始めましたが、それまでに蓄積されたノウハウがリセットされ、継承されずに同じ轍を踏んでいる場面もよくみられるようになりました。

会場を押さえたのは良いが、受付のシステムが難しく入館で人が詰まる。申請不備でネットが使えず、入館証が表示できない。

出入りに社員が張り付く必要があり、導線が悪く参加者の移動が休憩時間に終わらない、

パネルディスカッションで、それっぽい人をとりあえずずらっと並べたが、導入を話して盛り上がり切る前に時間が来てしまう。

ケータリングを発注したが、荷下ろしのための駐車場を申請しておらず、一本先の通りから担いで運びこむ。

養生してないために、台車が使えず担いで上がる。発注しすぎて残った残飯の処理を考えておらず、持ち帰ることになる。

新しくカンファレンスを立ち上げ、多額のスポンサーやチケット代を集め、海外登壇者の招致や、同時通訳、グッズやサイトの作成で予算を使ったが、当日大雪で中止になり、チケットもスポンサーも全額返金で主催が一気に 1000 万くらい借金背負う。

そうしたミスは、知識ある状態で主催すれば、全て避けることができる失敗です。

このドキュメントは、カンファレンスを開催する上で注意すべき点を有志でまとめ、主にこれからイベントを主催するスタッフの間で共有できることを目的にしています。
